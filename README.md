BookManager Project

This is a **C++ Book Manager** project created in **Visual Studio 2019**.  
It allows you to manage books with basic operations such as adding, updating, displaying, and storing books.

---

#Features

- Add a new book with ID, title, author, and quantity  
- Update book details or quantity  
- Display all books in a formatted table  
- Input validation for integers and strings  
- Serialize book data to a string format  

---

#Technologies Used

- C++ (C++11/14 standard)  
- Visual Studio 2019  
- Standard Template Library (STL): `vector`, `unordered_map`, `shared_ptr`  
- Header (`.h`) and source (`.cpp`) file structure  


---

## **How to Run**

1. Open `BookManager.sln` in *Visual Studio Code*  
2. Build the solution (Ctrl + Shift + B)  
3. Run the program (F5)
